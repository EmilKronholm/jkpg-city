This website was created by Emil Kronholm as a school project for the great course Web Development - Advanced Concepts, held by Patrick Riehmann at JTH in JÃ¶nkÃ¶ping. The course focuses on theoretical in depth concepts such as web history, cookies, event propagation and rendering. And combined we will also work with low-level native technologies (e.g. no frontend frameworks, just using DOM-manipulation).

This website is static and is completely rendered on client side using REST-API (with JWT authentication) for dynamic content.

The webserver is static and hosted with nginx. It's optimized for static websites, event-driven and as a result; extremely fast. And by custom config, we can still have protected routes with built in authentication. We can also very easily structure our project however we'd like (e.g. using seperate folders and subfolders for HTML/CSS-files, which isn't always straight forward on static webservers) using the try_files keyword.

We're only using CSR for dynamic content (a requirement for the course) which honestly I've had a bad experience with during this project.
